# Intelligent Document Workflow System

## Overview

Intelligent Document Workflow System is a Python-based project designed to simulate an intelligent administrative management platform for an electricity company. The system automates document reception, classification, prioritization, routing, notifications, and reporting processes through a multi-agent architecture.

The objective of this project is to demonstrate how intelligent systems and workflow automation can improve administrative efficiency and optimize document processing operations.

This project was developed as part of an academic and practical exploration of intelligent systems, software architecture, and automation using Python.

## System Architecture

The system is composed of multiple specialized agents responsible for different administrative operations.

### Workflow

```text
Document Reception
        ↓
Content Analysis
        ↓
Document Classification
        ↓
Priority Detection
        ↓
Department Assignment
        ↓
Notification & Reporting
        ↓
Archiving and Tracking
```

### Main Components

```text
- Reception Agent
- Classification Agent
- AI Decision Agent
- Notification Agent
- Alert Agent
- Tender Management Agent
- Reporting Agent
- Simulated Database
```
